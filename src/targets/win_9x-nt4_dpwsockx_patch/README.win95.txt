To run the `win_9x-nt4_dpwsockx_patch.exe' program, Windows 95 users may need
to install the Microsoft Visual C++ 6.0 run-time components. It, among others,
contains the C run-time library `msvcrt.dll' against which the
`win_9x-nt4_dpwsockx_patch.exe' program is linked.

See http://support.microsoft.com/kb/259403 for more information.
